# v1.0.0 — Initial release

**Date:** 2026-03-02
**Tag:** `v1.0.0`
**Events:** 73

First public release of the **Worldwide Events** indicator. Overlay key
historical events directly on the price chart and see how the market reacted.

## Features

- 📅 73 historical events plotted as vertical lines + labels on any chart
- 🎨 Three color-coded priorities — 🔴 high / 🟠 medium / 🟡 low
- ⚙️ One-click category filters (show only the priorities you care about)
- 🌐 Two languages for event descriptions — 🇬🇧 EN and 🇵🇱 PL
- 🔍 Hover tooltip shows the exact date without cluttering the chart
- 🔗 Each event links to its source (mostly Wikipedia)
- ⚡ Zero configuration — works immediately after adding to a chart
- 🆓 Open source (MPL 2.0)

## Notes

Event data lives in `data/events-en.md` / `data/events-pl.md`; the Pine Script
file is generated from it by `scripts/generate-events.py`.
